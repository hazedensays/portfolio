UI 및 기술 소개
🔎 Main 검색기능

Main 페이지 상단바 검색 기능

😎 DB(product table)에 등록되어 있는 제품을 검색했을 시에
검색된 내용을 포함한 모든 제품을 보여주며
해당 제품 클릭 시에 상세 페이지로 이동할 수 있도록 구현

😎 검색된 내용이 없을 시에는 "검색 결과가 없습니다."라는 창을 띄움

🔎 New

신상품 진열 및 이미지 슬라이드

😎 DB(product table)에 등록되어 있는 제품을 기준으로
제일 마지막에 등록된 4개의 제품을 띄우며
해당 제품 클릭 시에 상세 페이지로 이동할 수 있도록 구현

😎 Swiper 모듈을 사용하여 슬라이드 구현 후
해당 사진에 상품ID(key) 값을 주어 상세페이지로 이동하도록 구현

🔎 Cart

장바구니 담기, 수량 변경, 삭제 및 주문

😎 기존에 장바구니에 담긴 상품은 수량을 누적하여 카운팅하고
장바구니에 처음 담는 상품은 상세페이지에서 설정한 수량만큼 카운팅하도록 구현

😎 장바구니 내에서 수량 수정 가능

😎 일부 선택 또는 전체 선택한 제품을 삭제 또는 주문할 수 있도록 구현

🔎 CS-1

FAQ, 1:1문의 등록, NEWS

😎 FAQ를 SearchBar를 이용하여 검색하고 모달창의 형태로 보여줌

😎 모든 항목에 유효성 검사를 실시한 1:1 문의 작성 후
모달창의 형태로 작성한 게시물 확인이 가능하며 수정 또한 가능하도록 구현

😎 FAQ 와 NEWS 또한 관리자 페이지에서 글 등록 시 업로드 되도록 구현

🔎 CS-2

답변 확인 여부, 게시물 삭제

😎 관리자 페이지에서 해당 게시물에 답변을 남겼을 시에
해당 게시물 답글 여부에서 "N"이 "Y"로 바뀌도록 구현

😎 각각의 게시물 옆에 삭제 기능 구현

🔎 비밀번호 찾기

고객 Email로 랜덤 비밀번호 발송

😎 고객이 DB(User table)에 등록되어 있는 정보를 입력했을 경우
고객이 입력한 이메일 형태의 아이디로 8자리의 랜덤 비밀번호를 발송함

😎 이메일 발송 후 alert창을 띄우고 login 페이지로 화면 전환
